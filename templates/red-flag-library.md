# Red Flag Library

Copy and adapt these findings. Do not paste them unchanged unless they fit the project.

## Critical

### Hidden Ownership / Reserve

Finding:

```text
The launch does not explain who holds supply outside liquidity or public sale. This creates immediate trust risk.
```

Fix:

```text
Publish reserve wallet, allocation, allowed uses, and movement rules before launch.
```

### Fake Endorsement

Finding:

```text
The copy or visuals imply affiliation with a creator, brand, platform, or community without proof.
```

Fix:

```text
Remove the reference and add plain no-affiliation language.
```

### Manipulation Language

Finding:

```text
The launch copy encourages coordinated price action.
```

Fix:

```text
Replace price instructions with factual launch information and risk language.
```

## High

### No Authority Status

Finding:

```text
Mint, freeze, or update authority status is not disclosed.
```

Fix:

```text
Publish authority status and explorer links. If an authority is retained, explain why and for how long.
```

### Blank Or Mismatched Socials

Finding:

```text
Official links are missing or inconsistent across token page, website, X, and Telegram.
```

Fix:

```text
Create one official-link source and make every page match it before launch.
```

### Profit Claims

Finding:

```text
Public copy implies buyers should expect gains.
```

Fix:

```text
Remove outcome language. Use education, meme, culture, or experiment language only.
```

## Medium

### Weak X Bio

Finding:

```text
The X bio does not explain what the project is or what it is not.
```

Fix:

```text
Add one-line concept, chain/context, no-affiliation if relevant, and no-promise language.
```

### Unclear Telegram Rules

Finding:

```text
Telegram does not warn against scam links, DMs, fake support, or price promises.
```

Fix:

```text
Pin rules and official links before launch.
```

### Visual Reads Poorly Small

Finding:

```text
The token image is not readable at small avatar size.
```

Fix:

```text
Simplify the image, increase contrast, and remove tiny text.
```

## Low

### Vague CTA

Finding:

```text
The site CTA is too vague.
```

Fix:

```text
Use direct CTAs such as "Read Risk", "View Official Links", or "Join Telegram".
```
